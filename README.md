# AVX Marketing Website

## Descrição
Website institucional da AVX Marketing, desenvolvido com React, TypeScript e Vite.

## Pré-requisitos
- Node.js (versão 16 ou superior)
- npm ou yarn

## Instalação

1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/avx-marketing.git
cd avx-marketing
```

2. Instale as dependências
```bash
npm install
# ou
yarn install
```

## Rodando o Projeto

### Desenvolvimento
```bash
npm run dev
# ou
yarn dev
```
Acesse `http://localhost:5173`

### Build para Produção
```bash
npm run build
# ou
yarn build
```

### Preview da Build
```bash
npm run preview
# ou
yarn preview
```

## Deploy

Opções de deploy:
1. Netlify
2. Vercel
3. GitHub Pages
4. Hospedagem tradicional

### Configurações Recomendadas
- Usar Node.js 16+
- Comando de build: `npm run build`
- Pasta de output: `dist`
